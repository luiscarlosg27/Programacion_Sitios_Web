Principales etiquetas HTML

Etiqueta Descripción
Encabezados Los encabezados estan definidos por la letra H seguido por la prioridad del encabezado siendo h1 el encabezado principal y h6 el encabezado de menor importancia o subtitulos.
Parrafos Los parrafos estan definidos con la letra P en HTML.
Vinculos Para insertar vinculos estos estan definidos por la letra A seguido por el atributo href y el vinculo.
Imagenes Para insertar imagenes se debe utilizar la etiqueta IMG seguido del atributo src y despues la direccion o ubicacion del archivo de imagen. Podras utilizar tambien el atributo alt para dar un texto alternativo si no se visualiza la imagen y podras definir las dimensiones con los atributos width y height.
Tablas Para insertar tablas se usara la etiqueta TABLE y dentro de ella se insertar las etiquetas th para definir el encabezado de la tabla y tr para definir la fila y dentro de esta se ingresara el texto de las tablas con la etiqueta. td
Listas Las listas estaran dadas por listas ordenadas se crearan con la etiqueta OL y listas desordenadas se crearan con la etiqueta UL, en ambas dentro de dichas etiquetas crearas los items con la etiqueta li.
Iframe El Iframe es usado para insertar paginas, documentos, videos y otros tipos de archivos usando la etiqueta IFRAME. Puedes usar atributos como title para agregar una descripcion del archivo, height y weigth para dimensionar la inserción.
SVG La etiqueta SVG es un contenedor para graficos SVG es decir Graficos de Vectores Escalables, la etiqueta usada para emplearlo es SVG seguido de los atributos para dimensiones.
Video Esta etiqueta nos permita mostrar videos en nuestro sitio web, sin embargo dentro de esta se puede utilizar el apoyo de la etiqueta source para poder insertar mas de una alternativa de formato de video.
Audio La etiqueta Audio nos permite realizar la insercion de audios en las paginas web, podemos utilizar el atributo source para agregar diferentes formatos de audio.

Los diferentes estilos CSS que se pueden aplicar

Estilo CSS Descripción Etiquetas
Color CSS Se puede aplicar estilos de colores a fondos, textos, bordes, se pueden utilizar diferentes valores para asignar color como nombre de colores, valores RGB, valores hexagesimal y otros.
  • background-color: blue
  • color: #55ff00
  • border:rgb(155,100,100)
Background CSS Las propiedades de los fondos en CSS se pueden modificar para crear diferentes efectos tanto color, imagenes, duplicacion de imagenes de fondo, fijación o inserción de fondos en diferentes posiciones.
  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position
Borders CSS Las propiedades de border en CSS permite especificar los estilos, el ancho, color de los bordes, estilos por cada borde y otros parametros que se pueden modificar con CSS.
  • border-style
  • border-width
  • border-color
  • border-radius
Margins CSS La propiedad de margenes en CSS crea un espacio alrededor de los elementos en el sentido exterior de los elementos.
  • margin
Padding CSS La propiedad padding de CSS permite establecer el espacio alrededor de un solo elemento como un texto.
  • padding
Height y Width CSS La propiedad height de CSS permite establecer el ancho y el alto de un elemento, estan no incluyen las propiedades de margenes, bordes o rellenos.
  • height
  • width
Fonts CSS La propiedad Fonts CSS definen la fuente, el tamaño y el estilo del texto.
  • font-family
  • font-style
  • font-size
Modelo de Caja CSS El modelo de caja es aplicando con varias propiedades CSS los cuales son el margen, el borde, el relleno y por ultimo el contenido Ejemplo:
div { width:100px; border:10px solid blue; padding: 20px; margin: 10px; }
Link en CSS Puedes aplicar propiedades CSS para cambiar el aspecto de los links. Ejemplo:
a:link, a:visited { background-color: #f44336; color: white; padding: 14px 25px; text-align: center; text-decoration: none; display: inline-block; }
a:hover, a:active { background-color: red; }
Position CSS Puedes aplicar propiedades CSS para cambiar la posicion de un elementos como estatico, relativo, fijo, absoluto
  • static
  • relative
  • fixed
  • absolute
  • sticky
